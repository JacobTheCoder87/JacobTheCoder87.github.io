Credit Risk Model Stability TLab In this Home Credit - Credit Risk Model Stability project we are tasked with analyzing a person's ability to repay their loans with little to no credit history. This way people who have been traditionally denied from a loan based on their credit history has a chance to still take out a loan. This is a group project between Carlos Martinez Abrego, Jacob Lam and Ramon Mora.
Financial Fraud Detection TLab In this project, we are analyzing a large dataset for financial banking transactions. Our goal is to identify and flag as many fradulent cases as possible, while minimizing false positives as much as we can. The challenge of this project is the class imbalance of our dataset, where there would be an overwhelmingly more non-fraudulent transactions than there are fradulent.
Caesar Cipher TLab The purpose of this project is to program a code that will help to encode and decode a message when providing a word and a numeric key.
Rental Pricing TLab The purpose of this project is to utilize Jupyter Notebook and imported libraries, such as pandas, seaborn and sklearn linear regressions to analyze property prices in Taiwan.
Stock Data Engineering TLab The purpose of this project is to calculate the average, median, and standard deviation for Amazon's stock prices over 9 weeks. I tested the code using the validation script.
Shopping Behavior TLab The purpose of this project is to extract insights that will facilitate an international company's expansion into the US market. The focus lies in identifying the most frequently purchased items and their associated attributes across different seasons.
